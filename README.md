# shop

#### 项目介绍
使用主要技术： springboot， springdata jpa, shiro, echars, 图片上传

过程： 
1： 完成使用springboot整合shiro，登录过程完成验证，并且不同的用户，去到不同的页面，不同的角色有不同的权限

2： 交易网站的所有商品图片，都是上传到腾讯云的存储桶

3： 数据访问层，主要是用了springdata jpa

4： 交易平台，还引入了用户信用积分
